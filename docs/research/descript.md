## 2. 核心流程逐步还原

```
录制/导入 → 转写 → 文本编辑 / Underlord 指令 → 场景与精修 → 导出/发布
```

1. **输入**：上传（50GB/5K/ProRes）；应用内录制（屏幕/多轨）；Rooms 远程录制（10 人 4K 本地轨）；Zoom/SquadCast/Ecamm/Restream 一键导入；⚠️ **2025-12-05 官方关闭 YouTube 链接导入**（反爬）
2. **转写**：26 种语言（**官方口径：仅限拉丁字母语言，无中/日/俄**；列表已现希腊语/印地语 beta）；8+ 说话人检测
3. **编辑**：删字 = 剪片（品类开创者）；或 **Underlord** 自然语言多步编辑（模型选择器：Claude/GPT/Gemini 全系，2025-10 起，会话级切换）；时间线与场景编辑器精修
4. **再利用**：Underlord Create Clips 选段 + Center Active Speaker reframe + 动态字幕；Publish 生成社媒贴/博客/show notes
5. **导出/发布**：4K 本地导出；YouTube 直发 + 播客托管（Buzzsprout/Captivate 等 9 家）+ Wistia/Drive；**时间线导出** Premiere/DaVinci/Pro Tools/Logic；**无 TikTok/IG 原生调度、无 LinkedIn**

## 4. 深拆：Underlord 与模型选择器

- **模型选择器**（2025-10-03 上线）：Auto / Claude Fable 5 / Haiku 4.5 / Sonnet 5 / Opus 4.6-4.8 / GPT 5.4-5.5 / Gemini 3.1 Pro / 3.5 Flash；付费档解锁高级模型；一个选择器切换整个 agent 的 LLM 请求（按子任务自选是未来计划）
- **趋势意义**："bring-your-own-frontier-model"——把模型选择权交给用户/市场。对单模型架构的我们：短期不跟（体验一致性优先），但要在计费与能力叙事上回答"为什么我们不让你选模型"
- Underlord 能力面：去口头禅、缩词隙、去重拍、清晰度剪辑、Studio Sound、绿幕、眼神校正、自动多机位、Quick Design（自动场景/布局/B-roll）、Center Active Speaker、章节、Create Clips、Write（脚本/大纲）、Publish（社媒/博客/show notes）、图像/视频生成

## 5. 深拆：多语言栈（七家最强，但天花板清晰）

- **转写 26 语（拉丁字母限定）** → **字幕翻译 67 语** → **配音 28 语**（营销口径 30）："听懂窄、译出宽"的漏斗
- **勿译词表 + 转写术语表**：承认"机器翻译/转写必然错在专有名词"，用人工词表补洞——务实且有效；词表管理在 Brand Studio，勿译词表限 Business+
- **side-by-side 翻译校对视图**：翻译不是一键交付物，而是"AI 初稿 + 人校对"的工作流
- **对我们的意义**：这三点（校对视图、勿译词表、术语表）都应进入我们的多语言设计；我们的差异是**母语级文案质量 + persona 保持**，而非语言数量（26/67/28 vs 我们的 6 语深耕）

## 6. 信任伤口：2025-09 定价改制

- 2025-09-23/24 从"转写小时 + 无限 AI"切换为"媒体分钟 + AI credit"双表；老用户强烈反弹（有账单 $30 → $195/月案例；Trustpilot 3.0/5）
- **不滚存**为官方政策；失败任务扣 credit 为用户报告级（G2 投诉，官方无自动返还政策）
- **教训**：计量改制是对老用户的信任税；我们的计费应从第一天就"可预期、失败不扣费"

## 7. 一句话总结

Descript 是我们的交互北极星与多语言最强对手：它证明了文档式编辑与"AI 初稿 + 人校对"的专业工作流，但它的转写只有拉丁字母、文案是英文通用腔、数据在美国、LinkedIn 不存在——每一点都是我们的空位。
