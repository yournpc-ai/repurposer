## 1. 基础信息

| 项 | 内容 |
|:---|:---|
| 定位一句话 | 100+ 工具的生成式短视频工厂——prompt/URL/产品页/PDF → 竖屏成片，"replace 8+ tools" |

## 2. 核心流程

**输入（prompt/脚本/URL/音频/PDF）→ AI 生成（脚本+画面+配音 70+ 语言+字幕）→ 可编辑项目 → 直发 TikTok/IG/YouTube**
- **Auto-Mode Workers**（3-10 个）：定时自动生产的"内容流水线机器人"
- **生成模型超市**：按名暴露 Veo 3（100 credit）/ Sora 2（130）/ Seedance（200-1000）并明码标价
- 100+ 工具：头像/换脸/AI 播客/音乐生成（20 credit）/Suno-to-Video（导入 Suno 曲目）/教育解说/marketing 广告
- 声音克隆（Ultra）；3M+ 病毒视频 remix 库

## 3. 关键观察（对我们的意义）

1. **与我们定位正交**（生成式 vs 真实素材资产化），不构成直接竞争
2. **calculate-credits API**（官方文档确认）：渲染前免费成本预估，API/CLI/MCP 均可调——**透明计费的优秀实现，直接值得借鉴**（我们的"成本先告知"应对标它）
3. **失败生成照扣 credit**（Trustpilot 集中投诉；退款靠客服裁量，无自动返还政策）——反面教材
4. **Auto-Mode Workers** 的"定时自动生产"思想可借鉴为"会议后自动生成资产包"的自动化规则
5. 又一家"法国公司 + 美国数据"——EU 驻留空白的再确认

## 4. 一句话总结

Revid 是生成式工厂的激进样本：模型超市 + agent 基础设施 + 自动工人；对我们的价值不在功能对标，而在两个机制设计——成本预估 API（学）与失败扣费（戒）。
