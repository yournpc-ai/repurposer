# Agent Opus 调研卡

> **范式**: Factory（从零生成）· **公司**: Opus（与 OpusClip / OpusSearch 并行的独立产品线）· **更新**: 2026-07-21
> **证据**: 用户实测截图 ×16（composer / coachmark ×4 / 品牌库 / 模型选择器 / 灵感 gallery / Labs / 通知 / 用户菜单 / credits / projects / pricing）

## 1. 一句话定位

"描述创意 / 上传音频 → AI 生成故事视频"——从零生成 agent，对手是 Invideo / Pictory，不是 repurposing。**学交互手艺，不学产品方向**（生成工厂 = STRATEGY §1 判断 1 的商品化红海）。

## 2. IA 全图（"麻雀虽小五脏俱全"的拆解）

| 用户任务 | 承载 |
|---|---|
| 创作 | home composer（chip 三枚：风格·必需 / 资产·可选 / 声音·默认播音员） |
| 灵感 | 首页 gallery + 卡片 remix（混音版 / 样式按钮） |
| 管理 | projects（类型筛选 tabs：全部/视频/UGC/动态图形 + 搜索 + **日历视图 toggle，非路由**） |
| 身份配置 | 品牌库（侧边栏一级入口） |
| 计量 | sidebar credits 余额（450⚡）+ 分解 popover（一次性 390 + 月度循环 60） |
| 教育 | 通知中心（what's-new + 功能 CTA：品牌库 / MCP / 旁白预览） |
| 实验 | Labs 烧瓶位（AI UGC / 文章转视频 / Coming soon） |
| 账户 | 用户菜单（订阅 / 前往 OpusClip / 注销）；pricing 3 档 |

小而美的公式：**完整性 = 每个核心任务有一个可见的家**，不是功能数量。

## 4. 明确不拿的

- **模型超市分级转售**：Nano Banana Pro（图像）/ Seedance 1.5·2.0（视频，2.0 👑 绑 Pro 档）——矩阵 §I 💡 后排（否定超市形态，不否定多模型能力，ADR-025 已在 P1）
- **播音员音色**（"瑞秋"）：素材超市思路；我们是 Speaker 声纹（身份资产）
- **付费墙杠杆**：去水印 / 高级模型档 / 声纹·头像槽位（Pro 1 / Max 5）/ 排期器 Pro 墙——身份与排期在我们这不是 upsell
- **从零生成范式**：矩阵 §F AI 视频生成 💡 后排（persona 就绪前不进红海）

## 5. Pricing 结构（参照）

- **Free**：60 cr/月（= 2 个 30s 视频），完全编辑权限，预设 AI 语音库，定制徽标/角色/对象
- **Pro** $11.50/月（年付，原价 $29）：3600 cr/年（=120 视频）；去水印、**社媒排期器**、高级视频模型、头像/声纹槽位 ×1、多包购买
- **Max** $64.50/月（原价 $129）：18000 cr/年（=600 视频）；槽位 ×5、优先处理/支持、credits 购买 12% 折扣
- 年付六折 / 月付七折；credits = 一次性 + 月度循环分开陈列

## 6. 战略镜像

Opus = 广度对冲（三产品线 + 周级发版）；我们 = 深度对冲（身份 / 信任 / 机构闭环）。**不跟他们卷功能数量**（STRATEGY §3 牌 1）。
