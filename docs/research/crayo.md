## 2. 核心流程

**选模板 → 给主题/链接 → 生成 → 微调 → 下载自发**：
- 模板：Reddit 故事（粘贴 Reddit URL）、分屏 gameplay、假短信、ChatGPT 对话、Streamer 切片
- 生成：AI 脚本 + AI 配音（数百音色，高档声音克隆）+ gameplay 背景 + 15+ 病毒字幕样式
- 微调：轻编辑器（字幕/音色/背景/修剪）；**下载后自己发**——无发布、无 API、无品牌 kit、无团队

## 3. 关键观察（对我们的意义）

1. **与我们定位正交，基本不构成竞争**——它服务"无脸流量农场"，我们服务"真实演讲素材的知识资产化"
2. **官方白纸黑字 ALL SALES FINAL**（退款政策原文："does not offer refunds… under any circumstances"）+ 取消后扣费投诉——计费信任伤口的极端样本，反面教材
3. **模板即产品**：3 步向导、零学习成本——其"格式模板库"思路与我们"输出类型模板"（LinkedIn 长文/quote card）在商业结构上相似，但内容品味完全相反
4. 可借鉴仅限：极简向导式上手流程；不借鉴：病毒格式、无审核直出、退款政策

## 4. 一句话总结

Crayo 是"流量农场"生意的样本——增长凶猛但信任脆弱（实体不透明 + all sales final）；它的存在反衬出"面向机构与知识工作者"的产品必须在信任与合规上反向操作。
